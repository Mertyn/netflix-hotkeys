Netflix Hotkeys Help

    New keys

  • J, L Skip 5 seconds
  • K Play/Pause
  • Shift+ N Next episode
  • Shift+ C Cycle through subtitles
  • C Toggle subtitles on/off
  • V Cycle through audio
  • H Open/close this help menu

    Built-in

  • ←, → Skip 10 seconds
  • Space, Enter Play/Pause
  • ↑, ↓ Volume up/down
  • M Toggle mute
  • F Toggle fullscreen
  • Esc Exit fullscreen
  • S Skip intro/recap
